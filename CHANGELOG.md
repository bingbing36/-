## 迁移指南

### 从 v1 升级到 v2

1. 启动 Elasticsearch 服务
2. 更新配置文件，添加 Elasticsearch 信息
3. 运行初始化脚本：`python aibot-v2-elasticsearch.py --init-es`
4. 根据新的 API 更新调用代码

### 从 v2 升级到 v3

1. 配置 Tavily API Key
2. 更新系统提示词
3. 添加 `tavily_search` 到工具列表
4. 无需重新初始化 Elasticsearch

### 从 v3 升级到 v4

1. 安装 Gradio：`pip install gradio`
2. 使用新的启动脚本：`aibot-v4-gradio.py`
3. 界面和功能完全兼容，无需修改配置
